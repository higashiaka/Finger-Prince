# 👑 핑거 프린스 (Finger Prince)

> **"검색도 안 하고 질문부터 한다고요? 네, 이제 그러셔도 됩니다."**

핑거 프린스는 디시인사이드, 아카라이브 등 주요 커뮤니티의 흩어진 정보를 가장 빠르고 우아하게 찾아주는 **커뮤니티 특화 검색 클라이언트**입니다.

---

## 🚀 프로젝트 개요
커뮤니티는 정보의 보고이지만, 자체 검색 엔진은 가끔 우리를 실망시킵니다. 핑거 프린스는 유저가 '핑프' 소리를 듣기 전에, 아니, 핑프가 되어도 상관없을 정도의 **압도적인 검색 편의성**을 제공하는 것을 목표로 합니다.

---

## ✨ 주요 기능 (Roadmap)
- 🔍 **통합 검색**: 디시인사이드, 아카라이브의 여러 갤러리와 채널을 한 번에 스캔.
- 🛠️ **스마트 필터링**: 짤방 위주, 개념글/정보글 전용, 고닉/유동 필터링 등 강력한 옵션.
- 📖 **클린 뷰**: 광고와 불필요한 UI를 걷어내고 본문과 핵심 댓글만 빠르게 확인.
- 💾 **실시간 아카이빙**: 삭제될까 봐 걱정되는 꿀팁글들을 로컬에 즉시 저장.

## ⚠️ 주의사항
본 프로젝트는 정보 검색의 편의를 돕기 위한 도구입니다. 각 커뮤니티의 이용 약관을 준수하며, 과도한 트래픽 유발로 인한 차단 책임은 사용자 본인에게 있습니다.
